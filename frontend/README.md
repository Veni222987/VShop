# 若贝特购物网站项目

## 项目启动步骤

### 安装依赖
```sh
npm install
```

### 编译运行

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
### 新增文件夹用途
```
apis        -> API接口文件夹
composables -> 组合函数文件夹
directives  -> 全局指令文件夹
styles      -> 全局样式文件夹
utils       -> 工具函数文件夹
```
